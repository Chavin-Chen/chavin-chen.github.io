# 8.2 数值转换和随机

## 进制转换

```java
// ibase=2
Integer i = Integer.parseInt("1111011", 2);
// obase=16
String ans = Integer.toString(i, 16);
System.out.println(ans);
```

## 随机数

```java
Random random = new Random(System.currentTimeMillis());
// 返回 [0, 100) 的随机整数
random.nextInt(100);

// 返回 [0, 1) 的随机浮点数
double r = Math.random();
// 返回 [100, 200) 的随机整数
int ans = (int) (r * 100 + 100);
```
