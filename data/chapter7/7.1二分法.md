## 快速幂
求`x^y mod 10000007`的值。

```cpp
#define MOD 10000007
int pow(int x, int y) {
    if (y == 0) return 1;
    int t = pow(x, y / 2) % MOD;
    return (t * t * ((y & 1) ? x : 1)) % MOD;
}
```
