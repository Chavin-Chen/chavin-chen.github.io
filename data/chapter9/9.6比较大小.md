# 9.6 比较大小

```cpp
int biggerPos(int a, int b) {
    int c = 1;
    b = (a ^ b);
    if (isZero(b))
        return -1; // equals
    while (b >>= 1) {
        c <<= 1; // 为1的最高位
    }
    return (c & a); // 2^x
}

// 比较a,b大小
// @return a<b return 0
//         a==b return -1
//         a>b return positive integer(2^x);
int bigger(int a, int b) {
    if (isNegative(a)) {
        if (isNegative(b)) {
            return biggerPos(complement(b), complement(a));
        }
        return 0; // -a < +b
    }
    if (isNegative(b))
        return 1; // +a > -b
    return biggerPos(a, b);
}
```
