# 9.4 正负和绝对值

## 判断正负

```cpp
int isPositive(int a) {
    // 存在非0位，且符号为为0
    return (a & 0xFFFFFFFF) && !(a & 0x80000000);
}

int isZero(int a) {
    // 所有位都是0
    return !(a & 0xFFFFFFFF);
}

int isNegative(int a) {
    // 符号位为1
    return a & 0x80000000;
}
```

## 求绝对值

```cpp
int abs(int x) {
    int i = x >> 31; // 算术右移，取符号位
    return (x ^ i) - i;
    // return i==0 ? n : (~n + 1);
}
```
