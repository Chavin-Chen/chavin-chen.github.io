# 8.5 归并排序

1. 把序列分成两个子序列，递归下去直到子序列长度为1。
2. 按顺序合并序列到一起。

*注：*当数据规模扩大的时候，`快排 > 归并 > 希尔 > 堆`

```cpp
// 分治：归并排序
void mergeSort(int *arr, int left, int right) {
    if (left + 1 >= right) {
        return;
    }
    // 找到二分点
    int mid = left + (right - left) / 2;
    mergeSort(arr, left, mid);
    mergeSort(arr, mid, right);

    int i = left, j = mid;
    int *temp = new int[right - left], ap = 0;
    // 合并两个部分
    while (i < mid && j < right) {
        if (arr[i] < arr[j]) {
            temp[ap++] = arr[i++];
        } else {
            temp[ap++] = arr[j++];
        }
    }
    while (i < mid) {
        temp[ap++] = arr[i++];
    }
    while (j < right) {
        temp[ap++] = arr[j++];
    }
    for (int k = 0; k < ap; k++) {
        arr[left + k] = temp[k];
    }
    delete[] temp;
}

```
